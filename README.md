# Mini Canvas Editor

A mini design editor built using React and Vite.

## Features
- Add rectangles
- Drag elements on canvas
- Select elements
- Delete elements
- Duplicate elements
- Snap-to-grid dragging
- Text tool
- Export canvas to PNG
- Layer control (forward/backward)

## Tech Stack
- React
- Vite
- JavaScript
- HTML/CSS

## Run Locally

Clone the project

git clone https://github.com/Sharvani-06/mini-canvas-editor.git

Go to project folder

cd mini-canvas-editor

Install dependencies

npm install

Run development server

npm run dev
